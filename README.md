# odin-recipes
Hi everyone! This is my first project with TOP (The Odin Project). The goal for this project was to build a basic recipe website using only HTML. On the home page there's the main heading and then three links in an unordered list that takes you to the specific recipes page. Each recipe page has an image, description, ingredients, and steps. I decided to add a little more than what the instructions gave us so I added a way to return to the home page and I also added a way to go back to the top of a recipe page. I learned that you can add an ID attribute and use that as the URL in the href.
